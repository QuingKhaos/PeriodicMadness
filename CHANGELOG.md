# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Second third of content for tier 4.

### Changed

- [BREAKING] Move Blade Runners to tier 4 after Processing Units.

## [0.4.1] - 2026-02-09

### Added

- Compatibility with Awesome Sink Storage.
- Compatibility with Flex Splines.

## [0.4.0] - 2026-02-09

### Added

- Integrate Refined Power.

### Changed

- Replace Carbon Dioxide Gas with CO2 from Refined Power internally.
- Replace Steam with High Pressure Steam from Refined Power internally.
- Replace Water Vapour with Low Pressure Steam from Refined Power internally.

## [0.3.0] - 2026-02-08

### Added

- First third of content for tier 4.

## [0.2.2] - 2026-02-08

### Fixed

- Fixed animation crash on furnaces.

## [0.2.1] - 2026-02-07

### Added

- Conveyor meshes for some plates.

### Changed

- [BREAKING] Use different base game item for Bismuth Crystals.

## [0.2.0] - 2026-01-31

### Added

- Content for tier 3.

### Changed

- [BREAKING] Moved chlorine gas internally. You may have to reselect chlorine recipes in machines.
- [BREAKING] Moved coke internally. You may have to reselect coke recipes in machines.
- [BREAKING] Replaced Smelter with custom placeholder furnace using Foundry as base.

## [0.1.4] - 2026-01-27

### Fixed

- Add missing pipe output to assembling machine.

## [0.1.3] - 2026-01-22

### Fixed

- Make Nickel available in HUB Upgrade 3 already.

## [0.1.2] - 2026-01-22

### Fixed

- Crusher Mk.0 should require Iron Gear Wheels instead of Basic Motor.

## [0.1.1] - 2026-01-22

### Fixed

- Fixed missing is valid class check on optional dependencies CDO edits.

## [0.1.0] - 2026-01-20

### Added

- Content for phase 1.
